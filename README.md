# Rembrandt

GPU-accelerated image effects studio. Apply 25+ real-time visual effects to any image using WebGL2 shaders — right in the browser.

## Effects

| Artistic     | Classic        | Futuristic   | Distortion           |
|--------------|----------------|--------------|----------------------|
| Oil Painting | Grayscale      | Neon Glow    | Glitch Art           |
| Watercolor   | Posterize      | Synthwave    | Datamosh             |
| Halftone     | Emboss         | Wireframe    | VHS                  |
| Duotone      | Edge Detection | Infrared     | Chromatic Aberration |
|              | Pixelate       | Hologram     | Swirl                |
|              |                | Matrix       | Kaleidoscope         |
|              |                | Corrupt JPEG | Tilt Shift           |

Each effect has adjustable parameters (intensity, radius, threshold, etc.) with instant feedback.

## Run

```bash
bun install
bun run dev
```

## Stack

Svelte 5 · TypeScript · WebGL2 · Vite · Bun
